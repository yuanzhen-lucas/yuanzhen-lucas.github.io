R学习进程建议

初学者

没有一个起点适合所有初学者，但这里有 6 种开始学习 R 的方法。

像 tidyverse 一样安装、RStudio 和 R 包

​ 这三个安装步骤常常让初次使用的用户感到困惑。对于初学者友好的安装说明，推荐免费的在线 ModernDive 章节 R 和 RStudio 入门。您还可以享受 R-Ladies Sydney 的 Basic Basics 课程,该课程为新用户提供了 RStudio 的导览，以及安装和使用 R 包的分步指南。

花一个小时阅读 R 中整洁统计的简单介绍。

​ 如果您是从传统的点击式统计软件包（如 SPSS 或 SAS）开始使用 R，RStudio 的 Thomas Mock 创建了一个免费的视频网络研讨会，标题为 A Gentle Introduction to Tidy R 中的统计。这个一小时的介绍涵盖了如何快速开始使用 R 中的研究统计的基础知识，强调将数据读入 R，使用 tidyverse 进行探索性数据分析，使用 ANOVA 进行统计测试，最后使用ggplot2 做图。此外，您还可以通过该页面左侧的主题菜单找到许多其他 RStudio 网络研讨会和视频以进行探索。

阅读 R 了解数据科学。

​ 虽然视频对某些人来说很棒，但我们中的其他人通过蜷缩看一本好书来学习最好。如果这描述了您，请从您友好的当地书商那里获取 Wickham 和 Grolemund (2016) 的 R For Data Science 的副本。R For Data Science 提供纸质和免费电子形式，并已翻译成多种语言。 如果您对 R 的了解还不够，无法投身于 R for Data Science，您可能会发现 Garrett Grolemund 的 Hands On Programming with R (2014) 是一种更快的入门方式。它还提供纸质版、电子版和免费在线版本。

使用 RStudio.cloud Primers 开始编码。

​ 开始学习 R 的最有效方法之一就是开始使用它。 RStudio.cloud Primers 提供了一个基于云的学习环境，可以通过舒适的浏览器向您传授 R 的基础知识。 RStudio.cloud 不需要您在计算机上安装任何软件，让您可以轻松地用 R 轻松涉足数据科学领域。最重要的是，rstudio.cloud 帐户可免费供个人使用。

使用 R Markdown 发布您的作品。

​ R 是用图形和数据讲故事的绝佳工具，但有时也需要文字。 R Markdown 将叙述性文本和代码编织在一起，以生成格式优雅的报告、论文、书籍、幻灯片等。 Garrett Grolemund 将通过他的R Markdown 入门视频为您提供个人 R Markdown 之旅，或者您可以在 rmarkdown.rstudio.com 上选择自己的路径来探索 R Markdown 的奇迹。R Markdown: The Definitive Guide (2018) 它很好地概述了 R Markdown 系列软件包中的可能性。

了解一些用于开发的有效工具。

​ RStudio 提供了 6 个视频，称为RStudio Essentials 系列，可帮助您学习如何使用 RStudio 的工具（包括 RStudio 集成开发环境 (IDE)）对 R 项目进行编程和管理。这些视频还将帮助您学习良好的开发实践，使团队内的协作安全而轻松。

中阶学习者

一旦你开始学习 R，你可以通过探索 R 的许多专业能力来扩展你的技能。这里有 6 个最常见的领域，已经有一些 R 经验的人觉得学习这些特别有益。

拿一些备忘单（cheat sheets）。

​ 没有人能记住每个 R 包的所有函数和参数，这就是发明备忘单的原因。 RStudio 为最流行的 R 功能和包发布了免费的备忘单集合，以帮助记忆。如果你决定要收集它们，你可以克隆备忘单 github 存储库。

学会寻求帮助。

​ 每个人都会被卡住。学习在哪里以及如何寻求 R 帮助是一项需要磨练的强大技能。 Tidyverse 网站提供了一些关于如何寻求帮助您的专家性建议。你会喜欢的一个包是 reprex 包，用于创建可重现的 R 代码示例。通读reprex文章，其中包含大量动画 gif 以说明诸如Magic reprex 和 Using datapasta with reprex之类的步骤。在哪里寻求帮助？RStudio 社区是一个热情好客的在线讨论论坛，可以询问（并回答！）有关使用 R 的任何问题。

改善您的可视化。

​ 您可能已经知道如何使用 ggplot2 创建一个基本图形，您可以通过阅读 Hadley Wickham (2016) 的书 ggplot2：用于数据分析的优雅图形来扩展您对图形语法和 ggplot2 的知识。第三版正在进行中，可以在线免费查看，源文件在GitHub 上。建议也将 Winston Chang (2018) 更新的 R Graphics Cookbook加入书签，它解决特定 ggplot2 问题。

使用 htmlwidgets 和 Shiny 开发交互式应用程序。

​ 更好地传达您的分析的一种具体方法是使您的可视化具有交互性。您可以在 www.htmlwidgets.org上学习如何通过几行代码将基于浏览器的交互性添加到您的图形中。如果您的交互需求需要运行R代码，那么就需要在服务器上运行，请在 shiny.rstudio.com 学习如何编写 Shiny 应用程序，或者跟随 Wickham (2020) 编写新的 Mastering Shiny书。这两种方法都可以与 R Markdown 集成，以使用flexdashboard 包创建精美的交互式仪表板。

使用 tidymodels 简化您的模型探索。

​ 大部分数据科学都涉及建模，但每个建模包似乎都发明了自己的接口和参数。输入 tidymodels，这是一个用于建模和分析的元包，它共享 tidyverse 的底层设计理念、语法和数据结构。如果您以前使用过插入符号作为统一的建模界面，那么 tidymodels 包 parsnip 是它更新的子项。虽然该项目仍在开发中，但它有望显着简化模型探索。 RStudio 的 Edgar Ruiz 撰写了 tidymodels 的简要介绍来帮助您入门。

探索其他专业包。

​ R 吸引了数据科学家，因为它有 13,000 多个包，几乎可以解决所有用例。如果你对基因组学感兴趣，你会想花一些时间Bioconductor的包装集合。如果您在 Spark 集群上使用大数据，请查看 sparklyr。如果你想涉足金融，你可能会想从quantmod开始。要了解您应该探索哪些包，我们推荐一些基于主题的包目录，例如 Awesome R或 CRAN 任务视图。

高阶学习者

即使您是 R 代码开发方面的专家，深入学习一些议题既可以帮助您开发更好的代码，也可以更有效地与他人分享。

深入了解 R 的基础。

大多数 R 用户不是程序员，因此他们的大部分 R 代码没有应有的可读性、快速或高效。 Hadley Wickham 的书 Advanced R (2019) 的第二版解开 R 工作方式背后的许多秘密，并为您提供解决问题的新策略。您可能还想收藏Advanced R Solutions，它为本书中的练习提供了有效的解决方案。

了解如何扩展 R

虽然它的内容有时与 Hadley 的 Advanced R 重叠，但 R 核心团队在 CRAN 上提供了编写 R 扩展。如果您希望将自己的 C 和 C++ 例程添加到 R 中，这本书特别有用，而且还包含在其他任何地方都很难找到的关于调试、R API 和运行时分析的独特信息。

构建自己的包。

R 包允许您以模块化且易于集成的方式与其他 R 程序员共享您的函数。上面提到的 Writing R Extensions 记录了如何编写包，但是您可能会发现 Hadley Wickham 的R Packages book 更容易阅读.如果您希望您的 R 代码能够覆盖尽可能广泛的开发人员群体，您将想知道如何构建包。

在您的分析中使用 Python。

任何坚持你必须在 R 和 Python 之间进行选择来进行数据科学的人都是在创造一个错误的选择。 R 提供了一种将 Python 代码合并到 reticulate 包中的简单方法。 Sean Lopp 的网络研讨会R、RStudio 1.2 和 Python——一个爱情故事，展示了开发人员如何轻松地将 Python 代码集成到他们的 R 工作流程中，并逐步完成了网状 Shiny 应用程序的开发。

在 TensorFlow 上尝试深度学习。

RStudio 在 tensorflow.rstudio.com 上托管了一个专门用于 R 和 Tensorflow 的网站，您可以在其中学习如何在分析中使用深度学习。 J.J. Allaire 和 Francois Chollet 出版了一本使用R 进行深度学习的书。你可以听到 J.J.在他的 rstudio::conf 2019 视频中描述系统背后的理念。有关如何在 R 中开始深度学习的资源的快速概览，请阅读Sigrid Keydana 的文章并订阅 Tensorflow for R 博客。

用R Markdown 进行交流。

R Markdown 系列软件包使您能够创建和共享精美的数据科学产品，例如书籍、博客、网站和演示文稿。试用 bookdown、distill 和 blogdown 等软件包；每个都扩展了 R Markdown 以帮助您发布精美的网站以共享您的工作。如果您想展示使用 R 代码进行的分析或可视化，请考虑使用 xaringan 包使用 R Markdown 制作幻灯片。如果你已经构建了一个 R 包，那么pkgdown 包可以让你快速、相对轻松地为你的包构建一个网站，使用你已经完成的所有工作来编写和记录你的函数。
