非必要不重装（耗时，麻烦）

建议使用timeshift还原,Timeshift的官方配置

Timeshift备份与还原

系统备份

一般linux系统是比较稳定的，但是像袁振这样马马虎虎，又是一个Linux半吊子新手很容易出错，那么备份就显得十分重要，这里介绍备份的一种方法

此时你需要准备

  • 下载timeshift

    sudo apt-get update

    sudo apt-get install timeshif

  • 准备一个外接的磁盘（容量在32G以上），那一块磁盘通过外接式USB接口，搭配USB3.0来传输。

此时在你的Ubuntu本地软件库你就会找到Timshift这个软件图标，具体的可视化界面操作

系统还原

如果你的系统奔溃，但是有三种以下的情况，你可以分别应对

  • 如果能进入登陆界面，并且能够打开Timeshift，你就直接点击还原（restore），重启电脑就可。

  • 如果只能进入登陆界面，一般系统崩溃后不能进入桌面，但是能够进入登录界面，现象就是在登录界面输入密码后不会进入桌面，那么就要通过命令行的方式进行还原。

    • 通过·Ctrl+Alt+F1（一般F1-F6都可）进入tty终端

    • 输入用户和密码登录

    • 执行下面命令获取系统当前可以还原的节点

      sudo timeshift --list

    • 根据输出的内容选择一个节点进行还原(–skip-grub 选项为跳过grub安装，一般来说grub不需要重新安装，除非bios启动无法找到正确的grub启动项，才需要安装)

      sudo timeshift --restore --snapshot '2021-06-10_15-18-14' --skip-grub

    • 在输出的内容中一次输入【Enter】键和【y】键即可

      你不想输入这些的话直接使用

      sudo timeshift --restore --snapshot '2021-06-10_15-18-14' --skip-grub --scripted

  • 如果无法进入系统，通过U盘启动系统

    当登录界面和桌面环境都无法进入时，一般系统已经彻底崩溃，此时只能通过Linux live CD进行还原。

    • 制作Linux Mint启动盘
    • 进入Live系统后打开Timeshift软件，点击设置按钮，设置快照的储存位置
    • 后续步骤和第一和第二种情况相同按照线条的处理方式

tips

制作Linux Mint启动盘==（以下操作只是科普，未经过实际操作，慎行）==

  • 下载iso文件

  • 下载UltraISO ,傻瓜式安装，不用注册打开之后就进入主界面

  • 选择菜单栏中的【文件】，然后选择打开，这个步骤就是寻找你的镜像文件

  • 选择镜像文件，选择【打开】，我们就可以在主界面看到我们打开的镜像文件

  • 打开之后，选择菜单栏中的【启动】，【写入硬盘映像】

  • 在弹出的对话框中，选择【硬盘驱动器】，就是我们自己的U盘，我们插入u盘，就会自动显示。

    说明：制作U盘启动盘会导致U盘格式化，所以提前备份好U盘里面的东西，然后选择【写入】，就可以看到读条了！

  • 等进度条满格之后，就制作成功了！打开【计算机】，可以看到U盘也变了！

以上来源非官方的百度百科 可以详细看看官方文档

Live USB

Live USB 是USB随身碟或USB硬盘，里面含有完整的操作系统，可以被用来开机（booting）。Live USB很像live CD，但基本上有能力更改设定，而且可以把软件安装回USB装置上。像live CD一样，live USB可以使用在嵌入式系统系统管理（embedded systems system administration），资料还原（data recovery），或是不需要把操作系统安装到主机硬盘（local hard disk drive）里的测试。许多操作系统，包含微软的Windows XP Embedded和许多Linux套件和BSD可以被安装在随身碟上使用。差不多在你U盘上搞一个自己的小Linux系统

重装具体步骤

可以自行谷歌：利用U盘重装ubuntu系统（建议最新的稳定版），大致步骤为：

  • 制作启动U盘
  • 狂按键盘F12（或者开机会有提示，依据电脑型号而定）进入BIOS模式
  • 选择U盘启动，选择英语，并安装Ubuntu
  • 键盘布局默认，下一步，选择正常安装
  • 时区选择上海，下一步，你是谁？中的计算机名称建议简短（Transysbio）

仅供参考：使用U盘安装Ubuntu20.04
