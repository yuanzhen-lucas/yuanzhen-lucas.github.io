植物中的转录调控：利用组学数据破解顺式调控密码

​ 创新的组学技术、先进的生物信息学和机器学习方法正迅速成为植物功能基因组学不可或缺的工具，最近在该领域取得了巨大进展。在转录调控中，植物组学数据积累相对于动物数据的初始滞后刺激了能够从可用数据集中提取最大信息的计算方法的发展。最近对拟南芥和玉米中转录因子结合谱的综合研究以及公共数据库中统一处理的组学数据的积累使植物生物学家进入了大联盟，并提供了许多前沿方法。**Elena V.Zemlyanskaya等[1]**总结了用于预测或推断转录基因调控背后的顺式调控代码的最先进的生物信息学方法，重点关注它们的植物研究应用。

​ 染色质免疫沉淀测序 (ChIP-seq) 技术能够注释全基因组转录因子 (TF) 结合事件，而最近开发的 DNA 亲和纯化和测序 (DAP-seq) 方法 [2] 和一种有效的原生质体分离/转化系统，用于表达用于 ChIP-seq [3] 的表位标记 TF，使大规模 TF 结合分析成为可能。这些技术的应用揭示了大量拟南芥和玉米转录因子的结合特征。

​ 主要挑战是序列只是决定 TF 是否结合其靶 DNA 以及结合事件是否影响基因表达的众多参数之一。还必须考虑有关表观遗传标记、染色质可及性、TF 及其相互作用物的丰度、蛋白质-DNA 生物物理相互作用的特异性（对于不同成分的二聚体和多聚体可能是不同的）以及其他有助于触发调控的信息程序。尽管如此，基因调控区域中 TF 结合位点 (TFBS) 的检测是研究调控事件的先决条件，包括指导特定条件基因表达的 TF 的协同作用 [4]。

TFBS 模型整合了 TF-DNA 结合的结构特征，并作为另一种类型的输入数据。广泛使用的模型是基于词的（例如consensus或 k-mers）和基于轮廓的（例如位置权重矩阵、PWM）。尽管consensus模型仅粗略估计某些位点位置的变化，但对特定长度的所有可能序列（k-mers）的单独评分强化了基于单词的方法 [5]。 PWM 模型更精确地描述了 TFBS [4:1]；它是通过计算每个位置的核苷酸频率（位置频率矩阵）并将其转换为 PWM 以使所有位置的权重总和估计潜在 TFBS 的总分，从结合位点的对齐构建的。植物 TF 的频率矩阵冗余存储在多个数据库中（上表 ）。 基于词和基于轮廓的模型过度简化了 TF-DNA 相互作用的表示。用于构建 TFBS 模型的更高级方法考虑了结合位点内各个位置之间的依赖关系，例如使用隐马尔可夫模型算法 [6] 的 BaMM，TF-DNA 相互作用的生物物理特性，例如 BEESEM [7]，或本地DNA 构象，如 ShapeMF [8]。

​ 分析单个染色质沉淀峰集的标准步骤是从头基序发现(de novo motif discovery)，它使用例如 HOMER 工具 检测过度表示的序列模式（基序）。除了识别 TFBS 最常见的变体外，从头基序发现还揭示了其他过度代表的基序。这些可能包括 TFBS 的次要变体，反映了特定的监管模式。最近解决了发育调节因子 WUSCHEL (WUS) [9] 和 MONOPTEROS/生长素响应因子 5 (MP/ARF5) [10] 的 TF-DNA 复合物的晶体结构，为与次要和主要 TFBS 变体的差异结合奠定了结构基础.生长素信号调节因子 MP/ARF5 以不同的亲和力与 TGTCTC 和 TGTCGG 结合，芽分生组织组织者 WUS 与 TAAT 和 TGAA 结合。此外，替代基序可以描述伴侣蛋白的结合位点；例如，在花器官发生调节剂 AINTEGUMENTA (ANT) 的 ChIP-seq 峰中，基本戊半胱氨酸 (BPC) 和基本螺旋-环-螺旋 TF 基序的过度表达表明 ANT 可能被来自这些家族的 TF 募集到 DNA 中。 [11]]。

​ 二聚体和多聚体 TF 复合物在其多个靶标的调控区域留下独特的足迹。这些可以在生物信息学上推断为特定的顺式调控句法，即同型或异型基序簇具有一定的 TF 结合基序相互排列，有利于 TF 结合 。对于二聚体 TF 复合物，此句法表示二分（复合）顺式元件，由直接（尾对头、头对尾）、倒置（尾对尾）或外翻（头对头）方向重叠或用垫片分开 。在拟南芥 和玉米 中检测到来自 ARF 家族的激活剂和抑制剂的不同调节句法。尽管单体 ARF 激活剂和抑制剂结合相对相似的 DNA 序列，但 ARF 二聚体更喜欢具有不同结构的二分顺式元件 。

​ 预计许多植物 TF 会与 MP/ARF5 相互作用，因为它们具有共定位的结合位点 [12]；然而，在单个峰值集中对两个合作伙伴的 TFBS 的简单过度表示不足以推断交互作用。推断包含两个 TFBS 的语法、互惠 ChIP-seq 数据集中 TFBS 的共定位、同一组织中伙伴 TF 的共表达以及蛋白质-蛋白质和蛋白质-DNA 相互作用的实验测试是更好的选择。使用最先进的生物信息学方法，可以测试 TFBS 的共现并从单个 TF 绑定配置文件中推断语法。例如，SpaMo 工具检测用间隔物分隔的基序对的富集 。与将基序视为不可分割的序列块的 SpaMo 不同，MCOT 工具可以解释基序结构并识别重叠基序 。在植物中，已经描述了具有重叠基序的复合生长素响应顺式元件；例如，G 类 bZIP TF (GBF1/2) 和 ARF5 在调节血管发育方面的协同作用与其重叠的 TFBS 有关。

​ 检测与多聚体 TF 复合物相关的句法的生物信息学方法尚不可用；然而，二聚体结合的模型可以推广到分析更大的异聚体 TF-DNA 复合物 ，从而加强 SpaMo、MCOT 或其他类似工具在此任务中的应用。此外，全基因组 DNase-seq 和 ATAC-seq 技术识别与活性染色质相关的无核小体区域，使我们能够推断转录机制的复杂足迹 。在 ATAC-seq 的帮助下，已在拟南芥中研究了体细胞胚胎发生所必需的调控句法 [13]；对来自不同家族的 TF 的染色质共可访问性的分析表明，在此过程中 B3-TCP、ARF-B3、ARF-TCP 和 WRKY-CAMTA TF 对之间存在协同作用。

​ 为了从 ChIP-seq 数据中识别 TF 靶基因，有必要将峰位置与基因调控区域相关联。调节区主要位于转录起始位点的上游；然而，它们也经常出现在基因的下游、内含子、非翻译区，甚至蛋白质编码序列中。这种“非上游”调控区域在拟南芥的 TERMINAL FLOWER 1 (TFL1)、AGAMOUS-LIKE 6 (AGL6) 和 LEAFY (LFY) 花发育基因中被鉴定 。此外，在具有大基因组的物种中，例如玉米和大麦，远端调控元件（增强子）可能距离目标基因数十 kb ，这使得峰与基因的关联非常棘手。为了应对这些挑战，将表观基因组数据叠加到基因组注释上是有帮助的。例如，考虑到标记活性基因组区域的染色质可及性、组蛋白修饰和甲基化组数据，有助于查明功能性远端增强子并揭示它们在几种植物物种中的染色质特征 [14][15][16]。由染色质可及性和修饰数据组合确定的染色质状态全基因组图谱可用于拟南芥、玉米和水稻 [17]。组蛋白标记和 TF 结合谱的整合揭示了几种花主调节剂和发育动态增强剂中特定阶段 TF 的富集[18]。此外，高通量染色体构象捕获方法越来越多地应用于植物研究中，用于全基因组鉴定远距离增强子，这些增强子通过形成染色质环与靶基因的启动子相互作用。

​ 或者，将序列保守数据叠加在监管代码上，揭示了保守的顺式元件，因此可能对植物发育很重要。这种称为系统发育足迹的方法检测保守的非编码序列，可用于减少预测植物物种的假阳性顺式元件的数量。植物科学家可以选择许多预先计算的注释轨道，例如，PLAZA (https://bioinformatics.psb.ugent.be/plaza) 中可用的短保守非编码序列或 PlantRegMap [，在植物中，保守的调控区域比脊椎动物短得多，这使得很难检测远缘相关基因组中的保守性[19]。最近开发了一种无需比对的解决方案，即 CoMoVa 工具，用于在广泛的进化距离内检测已知 TFBS 中保守的小简并序列 [20]。

​ 基因附近的 TF-DNA 结合不一定会引起基因转录的变化。因此，测试顺式调节元件的存在是否与转录反应相关是至关重要的。耦合从 ChIP-seq 峰集推断的 TF 结合靶点和差异基因表达数据（例如，在敲除或过表达转基因系中由 TF 扰动触发）是区分调节性 TF 结合事件的直接方法[21] 。或者，瞬时分析报告转录因子（TARGET）的全基因组效应，控制TF核进入，测量TF诱导的基因表达变化，忽略二级TF靶标的下游调节； TARGET 可用作直接中等通量方法来识别 TF-DNA 调控对 [22]。使用一组可用的 TF 结合谱对差异表达基因的调节区域中的 TF 结合事件进行系统富集分析，可以预测上游调节因子的特定条件库 ；例如，多组学研究确定了茉莉酸 [23] 和脱落酸 [24]激素反应背后的调节机制的新成分。

​ 机器学习 (ML) 方法是从全基因组注释数据中推断出顺式调控代码的推理统计的一种有前途的替代方法 [25][26]。 ML 通过直接从功能基因组学数据集中学习转录调控特征，提供了一种无需假设的方法来模拟调控结果。考虑到众多特征的贡献，这种方法有利于预测调节性 TF 结合事件和基因表达。为此，ML 模型建立在训练数据（大量注释良好的全基因组数据）之上；然后在独立的测试数据上对其预测能力进行统计评估，以便可以常规应用已建立的 ML 模型从感兴趣的数据集进行预测。

​ ML 成功地解决了对象分类问题，例如，根据其调控区域中表观遗传标记的组成来识别活跃和非活跃基因。 ML 还用于估计基因表达水平，阐明各种因素的贡献 。 ML 已在植物中用于评估 TF 结合的组合效应 ，预测调控句法 [27]，以及评估对非生物胁迫的转录反应 。值得注意的是，对来自一个物种的数据进行训练的 ML 模型能够预测不同植物的转录结果，因为它已被证明可用于冷应激反应 [28]。

深度学习是一类使用能够自动提取特征的深度神经网络的 ML 方法，在预测解析基因组序列的调节结果方面非常有效（图 二）。最近，开发了一种深度学习模型来估计顺式调控序列与以碱基对分辨率检测 TF 足迹的 ChIP-nexus TF 结合谱之间的关系 [29]。该模型直接从序列中预测 TF 结合谱，为发现顺式调控语法提供了一种强大的方法。原则上，经过训练的神经网络有可能指导任何所需基因的基因表达工程[66]。虽然只有少数 ML 应用程序可用于推断植物顺式调控代码，但这些应用程序可能是下一个育种时代的关键，在该时代，作物性状将使用基因组编辑方法进行遗传改良 [30]。

​ 由于顺式调控代码非常复杂，并且在每种细胞类型中表现得非常具体，因此其综合重建的进一步进展需要来自模式植物物种的大量高质量数据。在这方面，类似于人类 ENCODE 项目的植物计划将是无价的 。单细胞多组学的快速技术进步为在单细胞分辨率下研究顺式调控模式提供了新的强大工具。最近已经进行了将单细胞转录组学和表观基因组学空间整合到二维细胞布局中的一个突出例子，以检查果蝇眼 - 触角盘的发育。最近出现的植物根部single-nucleus RNA-seq/ATAC-seq 数据 [31] 承诺将单细胞数据与植物领域中成熟的数学模型 [32] 相结合，以了解根部的形态发生过程干细胞生态位。最后，我们期待计算方法的进一步发展，以推断解释低亲和力 TF 结合 [33] 或多聚 TF 复合物结合的顺式调节代码元素。这种生物信息学方法将进一步提高我们推断影响基因调控的因素之间的调控句法和合作推断的准确性。
