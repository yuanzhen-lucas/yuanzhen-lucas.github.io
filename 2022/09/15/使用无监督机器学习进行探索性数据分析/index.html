降维技术：在 2D 中可视化复杂数据集

在统计学中，降维技术是一组通过获得一组主变量来减少随机变量数量的过程。例如，在跨不同患者样本的基因表达矩阵的背景下，这可能意味着获得一组新变量，涵盖基因组的变异。通过这种方式，样本可以由几个主要变量而不是数千个基因来表示。这对于可视化、聚类和预测建模很有用。

主成分分析（ Principal component analysis）

主成分分析 (PCA) 可能是检查高维数据的最流行技术。 PCA 如何降低维度有多种解释。我们将首先关注几何解释，该操作可以解释为旋转数据的原始维度。为此，我们回到我们的示例基因表达数据集。在这个例子中，我们将代表我们的患者只有两个基因的表达谱，CD33 (ENSG00000105383) 和 PYGL (ENSG00000100504)。这样我们可以在散点图中可视化它们。

PCA 旋转原始数据空间，使新坐标系的轴指向数据方差最大的方向。轴或新变量被称为主成分 (PC)，并按方差排序：第一个成分 PC 1 表示数据最高方差的方向。第二个分量 PC 2 的方向代表与第一个分量正交的剩余方差的最大值。这可以自然地扩展以获得所需数量的组件，这些组件一起跨越覆盖所需方差量的组件空间。在我们只有两个基因的示例中，主成分绘制在原始散点图上，在下一个图中，我们显示了基于主成分的新坐标系。我们将使用 princomp() 函数计算 PCA；此函数也返回新坐标。这些新坐标只是数据在新坐标上的投影。我们将用显示最大变化方向的特征向量来装饰散点图。然后，我们将绘制新坐标。这些是由 princomp() 函数自动计算的。请注意，我们在绘制坐标时以及在计算 PCA 之前使用了 scale() 函数。此函数将数据居中，这意味着它从向量中的元素中减去每个列向量的平均值。这实质上使列的均值为零。它还将数据除以居中列的标准差。这两个操作有助于将数据带到一个共同的尺度，这对于 PCA 不受数据中不同尺度的影响来说很重要。

在这种情况下，PCA 是通过称为特征分解的操作计算协方差矩阵的特征向量来获得的。协方差矩阵是通过我们的表达式矩阵的成对变量的协方差得到的:

cov⁡(X,Y)=1n∑i=1n(xi−μX)(yi−μY){ \operatorname{cov} (X,Y)={\frac {1}{n}}\sum _{i=1}^{n}(x_{i}-\mu_X)(y_{i}-\mu_Y)} cov(X,Y)=n1​i=1∑n​(xi​−μX​)(yi​−μY​)

用XXX和YYY表示的样本中基因的表达值，这是衡量事物如何一起变化的量度，如果样本 A 中高表达的基因在样本 B 中也高度表达，而样本 A 中低表达的基因在样本 B 中也低表达，那么样本 A 和 B 将具有正协方差(positive covariance)。如果相反，那么它们将具有负协方差(negative covariance)。这个量与相关性有关，相关性是标准化的协方差。可以使用 cov() 函数获得变量的协方差，并且这种矩阵的特征分解将产生一组正交向量，这些向量跨越最大变化的方向。在 2D 中，您可以将此操作视为将两条垂直线一起旋转，直到它们指向数据中大部分变化所在的方向。一个重要的直觉是，在特征向量规定的旋转完成后，这个旋转数据集中变量之间的协方差将为零。旋转数据集和原始数据集的协方差之间存在适当的数学关系。这就是为什么对协方差矩阵进行操作与原始数据集的旋转有关。

协方差矩阵的特征向量和特征值表示数据变化的方向和幅度。在我们的视觉示例中，特征向量是所谓的主成分。特征向量表示方向，特征值表示该方向的变化。特征向量和值成对存在：每个特征向量都有一个对应的特征值，并且特征向量彼此线性独立，这意味着它们是正交的或不相关的，就像我们上面的工作示例一样。特征向量按其对应的特征值排序，特征值越高，特征向量越重要，因为与其他特征向量相比，它解释了更多的变化。 PCA 的这一特性使得降维成为可能。我们有时可以仅以 2D 或 3D 显示具有许多变量的数据集，因为这些顶部特征向量有时足以捕获数据中的大部分变化。 screeplot() 函数将 princomp() 或 prcomp() 函数的输出作为输入，并绘制由特征向量解释的方差。

奇异值分解与主成分分析

计算 PCA 的一种更常见的方法是通过称为奇异值分解 (SVD) 的方法。这导致对 PCA 的另一种解释，称为“潜在因素”或“潜在成分”解释。稍后，我们所说的“潜在因素”是什么意思会更清楚。 SVD是分解输入矩阵的矩阵分解或分解算法，本质上，许多矩阵可以分解为多个矩阵的乘积。

UUU是在列上具有特征数组的矩阵，它与输入矩阵具有相同的维度；您可能会在其他地方看到这些列被称为 eigenassays(特征分析)，SSS是包含对角线上奇异值的矩阵。奇异值也称为特征值，它们的平方与每个特征向量的解释变化成正比。VTV^TVT(矩阵VVV的转置)包含其行上的特征向量。它的解释还是一样的。在几何上，特征向量指向数据中方差最大的方向。

上图中的一个新事物是特征数组的概念。特征阵列，有时称为特征分析，代表样本空间，可用于绘制样本之间的关系而不是基因。通过这种方式，SVD 使用协方差矩阵提供了比 PCA 更多的信息。它为我们提供了一种总结基因和样本的方法。因为我们可以将基因表达谱投影到前两个特征基因上并获得基因的 2D 表示，但是使用 SVD，我们还可以将样本投影到前两个特征数组上并在 2D 散点图中获得样本表示。如果我们有基于时间的表达谱，则特征向量可以代表跨样本的独立表达程序，例如细胞周期。但是，不能保证每个特征向量都具有生物学意义。类似地，每个特征阵列代表具有特定表达特征的样本。例如，激活了特定途径的样本可能与 SVD 返回的特征阵列相关。

以前，为了将样本映射到缩减的 2D 空间，我们必须在使用 princomp() 函数之前按样本转置基因矩阵。我们现在将首先在genes-by-samples 矩阵上使用SVD 来获得特征数组，并使用它来绘制降维上的样本。我们将把原始表达式数据中的列投影到特征数组上，并使用散点图中的前两个维度，这次我们还将使用 prcomp() 函数对转置的基因按样本矩阵执行 PCA，以获得相似的信息，并将样本绘制在简化坐标上。

如图看到的，这两种方法产生了样品分离，尽管它们略有不同。不同之处在于居中和缩放。在第一种情况下，我们缩放和居中列，在第二种情况下，我们缩放和居中行，因为矩阵是转置的。如果我们不进行任何缩放或居中，我们将得到相同的图。

特征向量作为潜在因素/变量

最后，我们可以通过 SVD 介绍 PCA 的潜在因子解释。正如我们已经提到的，在测量不同时间点采集的样本中的基因表达时，特征向量也可以解释为多个基因共享的表达程序，例如细胞周期表达程序。在这种解释中，表达程序的线性组合构成了基因的表达谱。线性组合只是意味着将表达式程序乘以权重并将它们相加。

这个简单的转换现在清楚地表明，如果特征向量代表表达程序，它们的线性组合构成了单个基因表达谱。例如，我们可以展示前两个特征向量的线性组合可以近似于基因表达矩阵中假设基因的表达谱，下图显示了特征向量 1 和特征向量 2 结合某些权重 WWW矩阵可以近似我们的示例基因的基因表达模式。

然而，SVD 并不关心生物学。特征向量只是从具有正交性和变化方向约束的数据中获得的。有代表真实表达程序的特征向量的例子，但这并不意味着特征向量总是具有生物学意义。有时它们的组合在生物学中可能比单个特征向量更有意义。这对于我们在下面描述的其他矩阵分解技术也是一样的。

其他用于降维的矩阵分解方法

继续介绍类似于 SVD 的其他技术。请记住，我们提到每个矩阵都可以分解为其他矩阵，其中矩阵乘法运算重建原始矩阵，这通常称为“矩阵分解”。在 SVD/PCA 的情况下，约束是特征向量(eigenvectors)/数组是正交的，但是，还有其他具有其他约束的分解算法。

独立成分分析(ICA)

我们将首先从独立成分分析 (ICA) 开始，它是 PCA 的扩展。 ICA算法分解给定矩阵:X=SAX=SAX=SA,

AAA(混合矩阵)的行类似于特征基因

SSS(元矩阵)的列类似于特征数组

这些成分在文献中有时被称为元基因(metagenes)和元样本(metagenes)

ICA 是针对称为“盲源分离”（blind-source separation）的问题而开发的，

举个例子：

多个麦克风记录来自多个乐器的声音，任务是从原始乐器中分离声音，因为每个麦克风都在记录声音的组合。

那么：矩阵SSS包含原始信号（来自不同乐器的声音）及其由权重标识的线性组合矩阵AAA，而一个矩阵SSS 和 AAA 组成矩阵XXX(来自不同麦克风的观察信号)。

考虑到这种解释，如果感兴趣的是表示隐藏表达程序的严格表达模式，我们会看到基因-样本矩阵被转置为样本-基因矩阵
